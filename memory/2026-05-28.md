## Lessons

- The audit subagent's first pass had some imagined line numbers; always confirm with `grep -n` and re-read the actual files before patching. Several "P0 missing accessibility label" claims turned out to be false positives because the code already had `accessibleButton`/`.accessibilityLabel` upstream.
- Dynamic Type safety is cheap: switching `.frame(height: X)` to `.frame(minHeight: X)` on text-bearing cards/charts removes a whole class of clipping bugs at large accessibility sizes.
- Token centralization reduces the audit surface area for the next release; future drift only needs `grep -n "cornerRadius: [0-9]"` and `grep -n "\.frame(width: [0-9]"` to enforce.
